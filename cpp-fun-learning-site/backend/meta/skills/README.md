# Skills

本目录存放项目内部技能说明。每个技能应独立成目录，并至少包含一个 `SKILL.md` 文件。

## 相关文档

- `backend/meta/skills/catalog.md`
- `backend/meta/skills/install-and-usage.md`
- `backend/meta/skills/baseline.md`

## 使用原则

- 技能用于沉淀可重复的工作方法
- 技能说明必须明确适用场景、输入、输出与限制
- 技能优先服务项目内部协作，不写入敏感信息

## 编写建议

1. 写清楚何时应使用该技能
2. 列出必读文件或依赖路径
3. 给出最小可执行步骤
4. 指出常见失败原因与回退方式
