# Init Full Template Command

            用于一次性初始化完整模板工作区。

            ## 目标

            - 写入基础目录
- 复制模板骨架
- 准备生成与正式文档目录

            ## 输入

            - 项目名称与 slug
- 目标目录
- 选定 profile

            ## 输出

            - 完整模板目录
- 模板状态文件
- 基础示例文档

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
