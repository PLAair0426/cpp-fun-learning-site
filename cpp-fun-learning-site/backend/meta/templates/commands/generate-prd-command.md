# Generate PRD Command

            用于基于输入材料生成产品需求文档。

            ## 目标

            - 梳理背景、目标、用户与场景
- 明确范围与非范围
- 形成 PRD 草稿

            ## 输入

            - 原始材料
- 业务目标
- 待确认问题

            ## 输出

            - PRD 草稿
- 待确认项
- 关键假设

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
