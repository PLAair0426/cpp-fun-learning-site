# Node CLI Command

            用于说明模板 CLI 的核心子命令与推荐用法。

            ## 目标

            - 列出常用命令
- 给出参数示例
- 说明初始化与同步顺序

            ## 输入

            - CLI 可执行入口
- profile 名称
- 目标目录

            ## 输出

            - 可复制执行的命令示例
- 命令说明
- 注意事项

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
