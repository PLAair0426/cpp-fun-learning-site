# Generate Execution Plan Command

            用于根据已确认的 PRD 与 Spec 产出执行计划。

            ## 目标

            - 读取 PRD 与 Spec
- 拆分阶段、任务与依赖
- 输出执行计划草稿

            ## 输入

            - 已确认的 PRD
- 已确认的 Spec
- 范围与优先级说明

            ## 输出

            - 执行阶段
- 任务拆解
- 风险与验证清单

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
