# Sync Template Command

            用于将模板更新同步到现有项目。

            ## 目标

            - 比较受管文件
- 预览变更范围
- 执行同步并保留受保护目录

            ## 输入

            - 模板版本
- 当前仓库状态
- 受保护前缀配置

            ## 输出

            - 同步清单
- 冲突项
- 回退建议

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
