# Bootstrap Project Command

            用于从模板骨架初始化新的项目目录。

            ## 目标

            - 创建基础目录结构
- 写入默认模板状态
- 根据 profile 初始化协作文档

            ## 输入

            - 项目名称、slug、目标目录
- 选择的 profile
- 是否写入默认模板状态

            ## 输出

            - 初始化后的目录结构
- 模板状态文件
- 基础说明文档

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
