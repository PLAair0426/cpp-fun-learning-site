# Generate Spec Command

            用于把 PRD 转换为技术与交付规格。

            ## 目标

            - 拆解模块边界
- 定义关键流程
- 明确接口、数据与依赖

            ## 输入

            - 已确认 PRD
- 技术约束
- 运行环境信息

            ## 输出

            - Spec 草稿
- 实现边界
- 风险清单

            ## 使用建议

            - 执行前确认规范与目录作用域
            - 复杂任务优先先生成草稿，再人工审阅
            - 涉及模板同步时，先预览再落盘
