# Document Templates

本目录保存正式文档模板，用于统一 PRD、Spec、执行计划与辅助记录。

在创建正式文档前，应先阅读 `backend/meta/specification/project-specification.md`。

## 主要模板

- `prd-template.md`：产品需求文档
- `spec-template.md`：技术与交付规格
- `execution-plan-template.md`：执行计划
- `decision-log-template.md`：关键决策记录
- `release-and-rollback-template.md`：发布与回滚计划
- `task-breakdown-template.md`：任务拆解表
